########################## COMPUTER ENGINEERING COURSE ########################
**Informações
|Foi desenvolvido pelos alunos do curso de Eng. Comp. da UFSM na disciplina de Sistemas Inteligentes
|Tem o jogo completo de Truco Gauderio
|O jogo tem a possibilidade de jogar humano x humano, humano x IA ou IA x IA
|A diferença dos Agentes esta na estratégia de jogo
|Os Agentes usam raciocionio baseado em casos para decidir as jogadas
|Baseado na base de dados sql que vem com esta pasta
|Todo os jogos são gravados em um arquivo .csv para analise (Data Mining por ex.)
|Para a IA foi usado o framework jColibri

*Passos
|Necessita de Java runtime 8 pelo menos
|É necessário importar a base de casos SQL (em um server mysql por exemplo)
|Usar o nome dbtrucocbrweb 
|ir no projeto cbr> hibernate> hibernate.cfg.xml alterar o password do mysql
|Usar o Eclipse
